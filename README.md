our tech blog

# how to install and use nuxt 
 1.  if you use macos or linux, do this cmd on your terminal $brew install yarn
 2.  if you didnt success $npm i -g yarn
 3.  $npm i -g @vue/cli @vue/cli-init
 4.  $vue -V
 

# how to start nuxt
 1.  go to proper dir
 1.  $npm init nuxt-app my-tech-blog
 2.  $cd my-tech-blog
 3.  $npm run dev
